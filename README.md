# 篮球记分牌

这是一个纯前端的篮球比赛记分网页，不依赖后端服务或数据库。比赛名称、球员、得分、犯规、操作记录和预设阵容都会保存在当前浏览器里。

## 直接使用

### 手机单文件版

如果球场边没有电脑，直接使用这个文件：

```text
basketball-scorekeeper-mobile.html
```

把它发到手机上，用浏览器打开即可。

### 微信链接版

如果需要通过 HTTPS 链接在微信里直接打开，使用这个文件：

```text
wechat-h5/index.html
```

上传到静态托管后，把链接发给记分员即可。

### 局域网访问

双击下面这个脚本，会在电脑上启动本地服务并生成手机可访问的链接：

```text
start-phone-link.bat
```

也可以手动运行：

```powershell
python -m http.server 8787 --bind 0.0.0.0
```

## 预设名单

默认示例名单放在：

```text
player-config.js
```

这里可以统一维护预设阵容中的球员名字和头像。当前仓库已经去掉真实姓名和头像，只保留了示例代称。

## 功能

- 设置比赛名称、日期、场地和阶段。
- 设置两队队名。
- 添加球员、删除球员、切换队伍。
- 记录每名球员的得分和犯规。
- 保存和加载预设阵容。
- 导出球员统计 CSV。
- 保存比赛记录 JSON，或使用记录码导出/导入。
